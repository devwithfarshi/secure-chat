# secure-chat

## Run Locally

Clone the project

```bash
  git clone https://github.com/devwithfarshi/secure-chat.git
```

Go to the project directory

```bash
  cd secure-chat
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
